# Personal_project1

내일배움캠프 6기 개인 미니프로젝트 23.05.28 - 23.05.31 해커톤

## 1. 프로젝트 소개

순수 바닐라 자바스크립트만을 이용하여 영화를 검색할 수 있는 서비스 만들기

## 2. 구현기능

- 영화카드 리스트 보여주기
- 영화카드 클릭시 영화 id를 나타내는 alert창 띄우기
- 검색
  - 일치하는 검색어 없을 때 => '검색결과 없음' alert, 페이지 리로드
  - 빈공백만 입력했을 때 => '문자입력하라' alert
  - 문자간 공백있어도 검색 가능
  - 대소문 구분 가능
- 최상단으로 가기 버튼
